# Timer Flashlight — AI-Assisted iOS Prototype

This project is a small experimental iOS application built to explore AI-assisted development workflows using Cursor agents and Figma MCP.

## Purpose

The goal of this project was not to hand-write production code, but to practice:

- Translating product ideas into clear prompts
- Guiding AI agents to generate working iOS code
- Reviewing, testing, and refining generated output
- Matching UI behavior and visuals to Figma designs

## How it was built

- Swift/iOS code was generated using Cursor through iterative prompting
- UI design and icon usage were guided via Figma MCP
- I acted as the system designer and reviewer rather than the primary code author

## My role

- Defined product requirements and user flows
- Designed and refined prompts for Cursor agents
- Reviewed generated code for correctness and usability
- Tested edge cases and refined behavior
- Ensured visual consistency with Figma designs

## What this demonstrates

- AI-assisted prototyping workflows
- Product thinking and UX judgment
- Ability to supervise and reason about generated code
- Rapid iteration and experimentation
